Detailed Description

template<class T>
class SimpleSet< T >

Very basic, light-weight unordered set for small data.

This uses a std::vector to hold the data and merely guarantees uniqueness (and immutability) of data and provides set-like methods. This has a smaller memory footprint and is faster than std::set for small amounts of data, but has O(n) complexity for insertion (all other operations are O(1)) so will perform worse than std::set for large n.
