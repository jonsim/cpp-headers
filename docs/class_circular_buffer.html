Detailed Description

template<class T>
class CircularBuffer< T >

Fast implementation of a Circular Buffer.

One space in the buffer is always left empty for technical reasons, so ensure that this is accounted for when the bufferSize is decided upon (i.e. maximumCapacity = bufferSize - 1). All operations on the Circular Buffer can be performed in constant time.
